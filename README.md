Este projeto foi desenvolvido por Bryan Machado da Costa e Silva, aluno de Sistemas de Informação da São Paulo Tech School, como parte das atividades orientadas pela professora Fernanda Caramico. O sistema tem como tema principal a Sociedade Esportiva Palmeiras, um dos clubes mais importantes do Brasil, e reflete sua paixão pessoal pelo time, que começou por influência de seu pai. Com o uso de tecnologias como Node.js, JavaScript e o MySQL, o projeto permite o cadastro e login de usuários, um quiz e uma dashboard para interação amigável com os dados, conectando o conteúdo técnico com a história e o sentimento de ser palmeirense.
